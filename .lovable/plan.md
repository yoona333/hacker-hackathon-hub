

# 🚀 AgentPayGuard - 3D 赛博朋克黑客松前端

## 项目概述
为 Kite Testnet 打造一个**极具视觉冲击力的 3D 赛博朋克风格**前端应用，结合 Three.js 粒子动画、霓虹发光效果和玻璃拟态 UI，让你的项目在黑客松中脱颖而出！

---

## 🎨 视觉设计

### 整体风格
- **深色主题**：纯黑背景 + 深蓝/紫渐变
- **霓虹配色**：紫色 (#A855F7) → 蓝色 (#3B82F6) → 青色 (#06B6D4) → 粉色 (#EC4899)
- **3D 元素**：交互式粒子星空背景、浮动几何体
- **玻璃拟态**：半透明卡片 + 背景模糊效果
- **发光效果**：霓虹边框、发光文字、脉冲按钮

---

## 📱 页面规划

### 1. 🌌 首页 Hero Landing
- **3D 粒子星空背景**（可交互，跟随鼠标移动）
- 大型发光 Logo + 项目名称动画
- **"Connect Wallet" 霓虹按钮**（脉冲发光效果）
- 三个核心功能卡片（悬浮发光）
- Kite Testnet 网络徽章

### 2. 📊 Dashboard 控制台
- **钱包状态栏**：地址显示 + 网络状态 + 余额
- **多签钱包信息卡片**
  - 合约地址（可复制 + 跳转浏览器）
  - 阈值进度条：2/3 多签可视化
  - 三个 Owner 地址（渐变头像）
- **冻结合约状态卡片**
  - 冻结地址数量（动态计数动画）
  - 合约状态指示灯
- **快速操作按钮组**

### 3. ❄️ 冻结/解冻操作界面
- **地址输入区域**（发光边框输入框）
- **状态查询结果**：冻结/未冻结状态卡片
- **操作面板**：
  - 冻结按钮（红色霓虹）
  - 解冻按钮（绿色霓虹）
- **已冻结地址列表**（滚动列表 + 状态标签）

### 4. 📝 多签提案管理
- **提案列表**（卡片式展示）
  - 提案 ID + 类型标签
  - 确认进度条（发光效果）
  - 确认者头像组
- **提案操作**：确认/执行按钮
- **新建提案表单**（步骤式引导）

### 5. 📜 交易历史
- **时间线布局**（霓虹连接线）
- 交易卡片：Hash + 类型 + 状态 + 时间
- 筛选标签组
- 空状态动画
